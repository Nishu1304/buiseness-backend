# Backend Setup

This project is a Django backend with Django Rest Framework.

## Setup

1.  **Activate Virtual Environment**:
    ```powershell
    .\venv\Scripts\activate
    ```

2.  **Run Migrations**:
    ```powershell
    python manage.py migrate
    ```

3.  **Run Server**:
    ```powershell
    python manage.py runserver
    ```
